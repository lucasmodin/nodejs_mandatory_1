Git basics

Git is a open source version control system. It allows developers to work on different feature, track changes, and collaborate efficiently.

Below are some common git commands that are useful for every project.

Get started

git init git clone &kturl>

Prepare to commit

git add <file> git add . git rm <file> git reset <file> git reset

Make commits

git commit git commit -m 'message' git commit -am 'message'
